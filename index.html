What's the Web's biggest problem?

Over 90% of web users in the UK are concerned about the ways that companies like Facebook, Google, and Amazon are amassing information about them -- but without alternatives, keep using these "cloud services" anyway.

The Web was designed to be democratic - let's take it back!

Introducing WebBox

WebBox is an open source platform that turns the web peer-to-peer. Information flows directly between people's WebBoxes - meaning no Centralized Databases or servers needed to share on the Web for the first time.

WebBox is built out of the web and uses many cutting-edge Web technologies - Linked Data, SPARQL, WebID, WebACL, REST and others.
